# TodoCondingExam
React.js で簡易的な TODO アプリケーション

![demo](image/image.png?raw=true "demo")

## 環境
Nextjs 14
Axios 1.6.8
MaterialUI 5.15.15
React 18
Tailwind-merge 2.2.2

## 仕様
1. 登録されているタスクを⼀覧形式で表⽰することができる
2. タスクを新規作成することができる 
3. 登録済みのタスクを更新することができる
4. タスクを削除することができる
